# Arman Mia | Developer Portfolio

A sleek, responsive, and performance-driven developer portfolio showcasing frontend web applications, technical expertise, and personal projects. Designed with a dark futuristic theme, fluid grid patterns, and dynamic UI elements.

🔗 **Live Demo:** [https://arman-mia.github.io/portfolio](https://arman-mia.github.io/PORTFOLIO-ARMAN-MIA/)

## 🚀 Features

- **Modern Tech Stack:** Developed completely using utility-first styling without bulky frameworks.
- **Dynamic Content:** Projects, skills, and academic background are generated dynamically via JavaScript data structures.
- **Fully Working Contact Form:** Integrated with Web3Forms API to deliver client messages directly to my inbox with custom animated toast feedback.
- **Direct Resume Download:** Allows users to easily download a copy of my professional CV instantly.
- **Fluid UI/UX:** Features interactive navigation scroll indicators, mobile responsive navigation drawers, custom scrollbars, and glassmorphic project cards.

## 🛠️ Built With

- **HTML5 & CSS3** - Semantic structuring and custom scroll/grid configurations.
- **Tailwind CSS (via CDN)** - Modern responsive layout design and gradient utilities.
- **JavaScript (ES6+)** - Dynamic DOM rendering, layout control, and contact form handling.
- **Lucide Icons** - Clean vector iconography across the page.
